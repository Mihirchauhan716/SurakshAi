# SurakshAI

SurakshAI is a lightweight AI-driven safety system designed to detect high-risk scenarios using motion and impact signals.

## Features

* Real-time anomaly detection
* Simulated ML-based risk inference
* Interactive demo visualization
* Confidence-based decision output

## How it Works

The system evaluates motion parameters such as speed and impact force to identify abnormal patterns.

For example:
A high impact force occurring at very low speed is flagged as an anomaly, as it deviates from normal motion behavior. This may indicate events like a sudden collision, fall, or device drop.
